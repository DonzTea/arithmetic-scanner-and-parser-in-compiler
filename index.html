Micro Lexical Analyzer

Scanner ini hanya menangani input operasi aritmetika seperti :
6x % 4 + 3y * 5xyz - 3z ^ 2 / 2

Batasan input :

  1. Menangani operator aritmatika berupa : +, -, *, /, %, dan ^
  2. Menangani variabel seperti : x, 2y, 5xyz, dan lainnya
  3. Token meliputi : angka, variabel, dan operator
  4. Tidak menangani delimiter seperti kurung buka dan kurung tutup
  5. Tidak menangani fungsi seperti : sin(), sqrt(), dan lainnya
  6. Tidak menangani operator logika seperti : and, or, >=, !=, dan lainnya


Hasil token akan muncul di sini setelah input diisi
